CanadaImmigration.py program displays the number of immigrants to Canada from 1980 to 2013 from a country specified by the User
In order to execute it, please follow the below steps:

1-Install the required libraries by running the below commands:

pip install pandas
pip matplotlib

2-Download the ImmigrationCanada.py program and the canadian_immegration_data.csv file and place them in a local folder

3-In the command line (or in your favorite IDE), type the command below:

py <<localfolder_path>>\CanadaImmigration.py

Where:
<<localfolder_path>> is the path in which you placed the CanadaImmigration.py file (i.e.: C:\MyPythonCodes)
