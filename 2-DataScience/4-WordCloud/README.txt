This program creates a "WordCloud" based on any subject queried on Wikipedia via its API. 

Running the program:

1-Download the code to a local folder
2-Navigate to the folder
3-Run the program with the command py ./createWorldCloud.py and when prompet, type any subject you want to browse
4-Go to the folder where the code is downloaded to see the wordcloud.png file with the WordCloud generated

Run the below commands to install the required libraries to execute the program:

pip install matplotlib
pip install wikipedia
pip install wordcloud
