CryptoRate.py program displays the 30 days historical prices for 1 BitCoin (BTC), Ethereum (ETH) or Solana (SOL), based on the User's selection.

In order to execute it, please follow the below steps:

1-Install the required libraries by running the below commands:

pip install pandas
pip install pycoingecko
pip install plotly
pip install mplfinance


2-Download the CryptoRate.py program and place it in a local folder
3-In the command line (or in your favorite IDE), type the command below:

py <<localfolder_path>>\CryptoRate.py

Where:
<<localfolder_path>> is the path in which you placed the CryptoRate.py file (i.e.: C:\MyPythonCodes)
